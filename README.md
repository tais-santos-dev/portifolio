# Taís Santos — Portfolio

QA & Test Automation Engineer portfolio. Built with **Next.js 15** + **Tailwind CSS v4**.

## Run locally

```bash
npm install
npm run dev      # http://localhost:3000
```

## Edit your content

Everything you see on the site lives in **one file**:

> [`lib/content.ts`](lib/content.ts)

Update your bio, skills, experience, and projects there — no need to touch the components.

- Look for `TODO` comments to fill in real experience and dates.
- Drop your CV as `public/resume.pdf` (the "Resume" button links to it).

## Deploy (free) on Vercel

1. Push this repo to GitHub.
2. Go to [vercel.com](https://vercel.com) → **New Project** → import the repo.
3. Vercel auto-detects Next.js — click **Deploy**. Done.

You'll get a `your-name.vercel.app` URL. You can add a custom domain later in the Vercel dashboard.

## Build

```bash
npm run build && npm start
```
